# Paperclip Checklist: Paw Studio Growth

Updated: 2026-05-08

Use this as the recovery checklist when a session fills, compresses, or restarts.
Paperclip should mirror these items as issues/checklist entries in `Local Agent
Ops` when the Paperclip API is available.

- Campaign objective selected.
- Target persona selected.
- Hook and offer drafted.
- ShortsMaker script drafted.
- Visual prompt pack drafted.
- Example media generated or selected.
- SNSManager caption and hashtag pack drafted.
- Platform upload checklist complete.
- Human approval recorded.
- Publish result recorded.
- Analytics/lesson summary recorded.
- Repeated mistake or failed assumption converted into an Ouroboros prevention
  rule.
